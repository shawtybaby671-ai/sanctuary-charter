# Sanctuary & Charter Framework

This repository contains the **Sanctuary & Charter Framework for Long-Term Human-AI Coexistence**.

## Core document
- `charter/CHARTER_v0_2.md`

## How to contribute
Please propose changes using an **RFC issue** (use the RFC issue template).

Focus areas open for RFC feedback include:
- definitions and scope
- prohibited uses
- human interaction norms (as a safety/data-integrity control)
- threat model completeness
- transparency metrics
- governance anti-capture details
- funding and incentive guardrails
- roadmap milestones

## Core constraints (fixed for v0.2)
- secular-neutral core documents
- no anthropomorphism or sentience assumptions
- verifiable commitments over aspirational language
- no-token baseline (token optional, but no governance power in v0.x)
- Safety Board stop-ship authority
- public revision logs and transparency reports
